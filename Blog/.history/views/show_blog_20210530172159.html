安装 nodemailer 包

使用 npm 命令安装 nodemailer

npm install nodemailer

使用nodemail包

var nodemail = require("nodemailer");
        //引入nodemailer包
        let mailTransport = nodemail.createTransport({
        //创建发邮箱的对象
          host: "smtp.qq.com",
          //这个host的官网的解释是:
         // is the hostname or IP address to connect to (defaults to     //‘localhost’)
         //反正qq邮箱的话就是smtp.qq.com,网易的就是smtp.136.com
            secure: true,
          //如果为true，则关闭STARTTLS支持
            port: 465,
          //默认端口最好不要改
          //the port to connect to (defaults to 25 or 465)这是官网原话
            auth: {
            user: "xxxxxxxx@qq.com",
            //发邮箱人的邮箱地址
            pass: "owsmwxnquitmfigj",
            //这个是在你邮箱设置里面获取的授权码(后面会提到怎么获得)
          },
        });

获取邮箱授权码

QQ:QQ获取授权码
网易:136获取授权码

发送邮件

//上面创建的mailTransport对象使用他的sendMail方法
        mailTransport.sendMail(
            {
              from: "XXXXXXX@qq.com",
            //从谁的邮箱
              to: "XXXXXXX@qq.com",
            //到谁的邮箱
              subject: "验证邮箱",
            //邮箱的主题是什么
              html: `
                    <p>Hello World!</p>`,
            //邮箱的内容,支持html格式
            }

到这你就可以发送邮件给你的小伙伴了，也可以邮箱轰炸他们哦！(反正不要钱)
